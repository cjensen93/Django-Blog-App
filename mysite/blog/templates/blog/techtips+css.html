HTML Elements

<img>

  • Purpose: Embed an image into a webpage
  • Required attributes: src
  • Other common attributes: alt, title, width, height
  • Display: Default is inline
  • Example: <img src="orange.jpg" alt="Picture of an orange"/>

<div>

  • Purpose: Container to help the flow of content
  • Required attributes: N/A
  • Other common attributes: class
  • Display: N/A
  • Example: <div class="thesis"></div>

<a>

  • Purpose: Create hyperlinks to other destinations on the internet
  • Required attributes: N/A
  • Other common attributes: href, download
  • Display: Default is inline
  • Example: <a href="https://www.amazon.com/">Amazon</a>

<h1>-<h6>

  • Purpose: Used as headers for websites, with various font sizes
  • Required attributes: N/A
  • Other common attributes: N/A
  • Display: Default is block-level
  • Example: <h1>Bertolli's Blog</h1>

HTML Character Entities

<

  • Name: Less Than symbol
  • How to use: &lt; or &#60;

>

  • Name: Greater Than symbol
  • How to use: &gt; or &#62;

&

  • Name: Ampersand symbol
  • How to use: & or &amp;

CSS Selectors

element

  • Purpose: It selects all elements of a given type on a page. Useful for general styling
  • How to Use: Using h1 { font-style: italic; } will make all the text within every h1 element italicized

.class

  • Purpose: Lets you assign different properties to multiple elements manually
  • How to Use: Using .style1 { background-color: blue; } can give many elements with the .style1 selector a blue background

#id

  • Purpose: Lets you assign a single type of element properties
  • How to Use: Using #id { color: skyblue; } will change the font color of a single element containing that ID to skyblue

:nth-child

  • Purpose: Can be used to assign properties to a group of siblings based on their position
  • How to Use: Using :nth-child(3n) { width: 200px; } would select every third child in a group of siblings to set the width to 200 pixels

CSS Properties

background-color

  • Purpose: Sets the background of an element to a specific color
  • How to Use: It accepts many values, including known common names (brown), rgb (150, 150, 100), rgba (100, 255, 255, 0.6), hexadecimal (#b36260), and many others

font-style

  • Purpose: Sets whether the font in an element are stylized
  • How to Use: It accepts the values of normal, italic, and oblique [degree].

padding

  • Purpose: Sets the spacing on all four sides of an element at the same time
  • How to Use: It accepts values of 10px (for all sides), 15% 20% (for vertical and horizontal), 2em 3em 2em (for top, horizontal, bottom), 15px 5% 2em 0 (for top, right, bottom, left)
